================================================================================
    MICROSOFT 基础类库: LayoutDesigner 项目概述
===============================================================================

应用程序向导已为您创建此 LayoutDesigner 应用程序。此应用程序不仅演示使用 Microsoft 基础类的基本知识，而且可作为编写应用程序的起点。

此文件包含组成 LayoutDesigner 应用程序的各个文件的内容摘要。

LayoutDesigner.vcproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件。
    它包含有关生成文件的 Visual C++ 版本的信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

LayoutDesigner.h
    这是应用程序的主要头文件。它包括其他项目特定的头文件(包括 Resource.h)，并声明 CLayoutApp 应用程序类。

LayoutDesigner.cpp
    这是包含应用程序类 CLayoutApp 的主要应用程序源文件。

LayoutDesigner.rc
    这是程序使用的所有 Microsoft Windows 资源的列表。它包括 RES 子目录中存储的图标、位图和光标。此文件可以直接在 Microsoft Visual C++ 中进行编辑。项目资源位于 2052 中。

res\LayoutDesigner.ico
    这是用作应用程序图标的图标文件。此图标包括在主要资源文件 LayoutDesigner.rc 中。

res\LayoutDesigner.rc2
    此文件包含不是由 Microsoft Visual C++ 编辑的资源。您应该将不可由资源编辑器编辑的所有资源放在此文件中。

LayoutDesigner.reg
    这是一个示例 .reg 文件，它显示了框架将为您设置的注册设置的种类。可以将它用作
    将与您的应用程序一起使用的 .reg 文件，或者索性将它删除，而采用默认的 RegisterShellFileTypes 注册。


/////////////////////////////////////////////////////////////////////////////

对于主框架窗口:
    项目包含标准 MFC 界面。

MainFrm.h, MainFrm.cpp
    这些文件包含框架类 CMainFrame，该类派生自
    CFrameWnd 并控制所有 SDI 框架功能。

res\Toolbar.bmp
    此位图文件用于为工具栏创建平铺图像。
    初始工具栏和状态栏是在 CMainFrame 类中构造的。使用资源编辑器编辑此工具栏位图，并更新 LayoutDesigner.rc 中的 IDR_MAINFRAME TOOLBAR 数组以添加工具栏按钮。
/////////////////////////////////////////////////////////////////////////////

应用程序向导创建一个文档类型和一个视图:

LayoutDoc.h，LayoutDoc.cpp - 文档
    这些文件包含 CLayoutDoc 类。编辑这些文件可以添加特殊文档数据并可实现文件保存和加载(通过 CLayoutDoc::Serialize)。
    文档将具有下列字符串:
        文件扩展名:layout
        文件类型 ID:Layout.Document
        主框架标题:LayoutDesigner
        文档类型名称:Layout
        筛选器名:Layout Files (*.layout)
        文件的新简称:Layout
        文件类型全名:Layout.Document

LayoutView.h，LayoutView.cpp - 文档的视图
    这些文件包含 CLayoutView 类。
    CLayoutView 对象用于查看 CLayoutDoc 对象。





/////////////////////////////////////////////////////////////////////////////

其他标准文件:

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 LayoutDesigner.pch 的预编译头(PCH)文件和名为 StdAfx.obj 的预编译类型文件。

Resource.h
    这是标准头文件，它定义新资源 ID。
    Microsoft Visual C++ 将读取并更新此文件。
